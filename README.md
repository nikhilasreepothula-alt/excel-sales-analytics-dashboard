
# 📊 Excel Sales Analytics Dashboard (AtliQ Hardwares)

## 🔍 Project Overview

This project is an Excel-based sales analytics dashboard built using a guided real-world dataset for AtliQ Hardwares.

The goal was to analyze sales performance, customer trends, and financial KPIs by building structured reports in Excel.

---

## 🎯 Goals

- Create a **Customer Performance Report**
- Compare **Market Performance vs Sales Targets**
- Build **Profit & Loss reports** by market, month, and fiscal year
- Analyze **Gross Margin % trends across quarters**

---

## 📌 Key Findings from Analysis

- Revenue increased significantly from FY2019 to FY2021  
- Gross margin percentage declined over the same period  
- FY2021 actual sales were below target by approximately 8%  
- Revenue is concentrated among a small group of customers  
- Certain markets contribute high revenue but show lower profitability  
- E-commerce channels show faster growth compared to other channels  

---

## 💼 How this helps business

- Supports tracking of sales performance across markets and customers  
- Helps identify gaps between actual performance and targets  
- Provides visibility into profitability trends over time  
- Assists in identifying key customers driving revenue  
- Helps improve reporting structure for decision-making
